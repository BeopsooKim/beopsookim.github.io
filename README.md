# Beopsoo Kim Academic Homepage

This repository hosts the GitHub Pages site for <https://beopsookim.github.io/>.

The site is based on the `academic-homepage` Jekyll theme and contains only public homepage materials. The downloadable CV is generated from the local reference-bearing CV by publishing pages 1-7 only; reference contact details are intentionally excluded from the repository.

The Inha University emblem used for affiliation entries is derived from the Wikimedia Commons file `InhaUniversity_Emblem.jpg`, which is marked as a public-domain text/logo asset with trademark restrictions.
